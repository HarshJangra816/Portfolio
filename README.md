# 🚀 Harsh Jangra – Portfolio

Welcome to my personal portfolio built with **React**.  
This project showcases my skills, projects, and achievements as a BCA student passionate about software development, data analytics, and game design.

---

## 📌 Features
- Modern responsive design (React + CSS)
- Project showcase with live demos and GitHub links
- Contact section with email and social profiles
- Easy to update and redeploy via GitHub + Vercel/Netlify

---

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, CSS
- **Deployment:** Vercel / Netlify
- **Version Control:** Git & GitHub

---

## 📂 Project Structure
src/
┣ components/   # Reusable UI components
┣ routes/       # Pages and routing
┣ assets/       # Images, icons, styles
